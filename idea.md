# SkillSync – AI-Powered Interview & Resume Preparation Platform

## Overview
SkillSync is a full-stack web application designed to help students and job seekers prepare for placements, internships, and interviews. It provides resume building tools, mock interview practice, skill tracking, and performance analytics in one platform.

## Problem Statement
Many students struggle with interview preparation due to scattered resources, lack of structured practice, and poor feedback on resumes and interview performance.

## Proposed Solution
SkillSync provides:
- Resume builder and storage
- Mock interview question generation
- Skill progress tracking
- Performance analytics dashboard
- Centralized preparation platform

## Key Features
- User authentication (JWT login/signup)
- Resume creation and management
- Interview practice module
- Skill tracker dashboard
- Analytics and performance reports
- Admin monitoring system

## Tech Stack

### Frontend

### Backend (Main Focus)
